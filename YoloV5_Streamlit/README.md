# YOLOv5 Object Detcetion Streamlit App

## Summary
An application that ultilizes YOLOv5 to detect objects in an image.

## Installation
``` 
    *clone the repository*
    cd YoloV5_Streamlit
    pip install -r requirements.txt
    streamlit run app.py
```
